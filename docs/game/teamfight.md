# 团战定位与切入

## Fizz 的团战角色

先明确一点：Fizz 在团战里**不是主力输出**，而是**刺客型侧翼威胁**。你的目标不是跟对面坦克打正面消耗战（你打不过），而是找到机会秒掉对方 C 位（ADC 或法师），然后用 E 或中娅全身而退。

如果你只记住一句话，那就是：**不要第一个进场**。

## 进场时机

### 理想进场时机

团战打起来后，双方的关键控制技能会在前几秒内大量释放（比如对面莫甘娜的 R、阿木木的 R、塞拉斐恩的 R 等等）。等这些大招交完之后，对方的 C 位就失去了保护伞，这时候你从侧翼或后方切入，效果最好。

### 进场检查清单

在按下 R 之前，快速过一遍这几个问题：
- 对方有没有保人大招还没交？（璐璐 R、卡尔玛 RE、守护者的大招等）
- 对方 C 位有没有闪现或位移？
- 我进去之后有没有退路？（E 或中娅能不能用？）
- 我的 R 能命中目标吗？（距离、角度、障碍物）

如果这些问题的答案都对你有利，那就是你的时刻。

## 切入路线

### 侧翼切入（首选）

最好的切入路线是从对方视野盲区绕到侧面或后方。利用 Fizz E 的翻墙能力，你可以从意想不到的角度出现在对方 ADC 面前。这需要你在团战开始前就提前卡好位置，而不是跟着大部队正面站一排。

### 正面切入（迫不得已）

如果没有侧翼的条件（比如地形不允许或者对方视野太好），你可以选择正面 R 然后 Q 进去。但这种方式风险很高——对方可以看到你的 R 并提前闪避或开保护。只在你非常确定能命中的情况下使用。

## 秒人之后怎么办

很多 Fizz 玩家的问题不是"切不到人"，而是"切到人之后就暴毙了"。秒掉 C 位之后你的处理方式取决于当前状态：

- **E 还在**：立刻 E 跳走，拉开距离等 CD
- **E 在 CD 但有中娅**：中娅按下去，等队友跟进
- **什么都没有**：那说明你进场前的检查清单没过好，这个问题要在复盘里解决

::: tip 后期团战的心理建设
后期团战非常紧张，你可能会犹豫"要不要进"。这种犹豫是正常的，但犹豫太久会让你的团队处于4v5的不利局面。**做好判断后果断行动，哪怕结果不完美，也比一直犹豫不进要好。**
:::
