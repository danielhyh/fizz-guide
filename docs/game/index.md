# 🎮 游戏理解

这里记录的是**不依赖于具体英雄**的通用游戏认知——无论你用什么英雄，这些东西都适用。

## 为什么单独拎出来

很多人把"游戏理解"等同于"会玩某个英雄"，但实际上这是两个独立的维度。你可以操作很好但宏观意识差（典型表现：对线杀穿但游戏总是赢不了），也可以操作一般但对地图的理解很深（典型表现：KDA 不高但胜率稳定）。

钻石段和翡翠段的核心差距，更多在游戏理解这一层，而不是在操作手速上。

## 内容板块

**宏观思路** — 游走、视野、团战定位，这些是把个人优势转化成团队胜利的关键桥梁。

**心态管理** — 排位焦虑、连败应对、长期成长心态。竞技游戏是心理战，稳定的心态本身就是一种实力。

---

> 💡 这部分的内容适用于所有英雄和所有位置，是真正的"底层能力"。
