# 视野控制

## 中单的视野责任

很多中单玩家觉得"插眼是辅助的事"，这在钻石段以上是一个非常致命的认知错误。中路是地图的枢纽，你的视野质量直接决定了你能不能安全游走、能不能提前发现对方打野的入侵。

## 关键插眼位置

### 进攻型视野（你有优势时）

当你对线占优的时候，视野的目的是**控制对方打野的动向**，让你可以更大胆地压线和游走。关键位置包括对方的蓝 buff 入口草丛、河道蟹位置、以及对面的红 buff 通道。

### 防守型视野（你劣势或平局时）

当你对线不占优的时候，视野的目的是**防止被 gank**。关键位置是你这一侧的河道草丛（两边都要关注）、以及中路两侧的过道草丛。

### 控制眼的使用

控制眼（粉眼）不要随便丢。最有价值的位置是**你需要长时间控制的区域**——比如你准备频繁游走的那一侧河道，或者即将开打的龙坑/峡谷先锋附近。

## 读图能力

插眼只是视野的一半，另一半是**从现有信息推断对方位置**。即使没有看到对方打野，你也可以通过以下线索判断他大概在哪：

- 对方哪一路突然打得特别凶？可能是打野在附近给他壮胆
- 对方打野最近一次出现在地图上是什么时候、在哪里？他接下来会去哪个野区？
- 兵线状态——如果你的下路把线推得很深而且没眼，那就是对方打野最诱人的目标

::: tip 钻石段的视野差距
翡翠和钻石的一个核心差距就在视野意识。翡翠段的玩家眼石经常满电荷，钻石段的好玩家眼石几乎随时是空的——因为他们一直在用。
:::

---

> 💡 视野控制的回报是隐性的，你可能感觉不到它救了你多少次，但它确实在无形中提升你的胜率。
