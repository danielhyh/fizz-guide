# 游走时机与路线

## 为什么游走是 Fizz 的核心竞争力

Fizz 作为刺客，单纯赢线不够——你需要把个人优势转化成全队优势。而游走就是这个转化的最高效方式。Fizz 的游走能力在所有中单里属于第一梯队，原因很简单：**E 可以翻墙**。

这意味着你的游走路线比绝大多数中单更灵活、更快、更难被对手预判。

## 游走的前提条件

不是什么时候都该游走的。盲目游走的代价是丢兵、丢经验、丢塔皮，如果还没拿到人头，那就是血亏。游走之前先检查这几个条件：

**兵线状态**：理想情况是你把兵线推到对方塔下再走。这样对手要么接兵放弃跟随，要么弃兵跟着你但损失经验金币。最差的情况是你兵线没推就跑了——对手不仅能跟，还能顺便发信号。

**对手位置**：如果对手消失了，你游走可能撞上他的反蹲。如果对手在线上老老实实补刀，那就是你的窗口。

**目标路线的状态**：你要去的路是否有击杀条件？队友血量够不够配合？对方是否推线推得很深？如果下路双方半血和平补刀，你跑过去也未必能打出什么。

## 游走路线选择

### 下路游走（最高优先级）

下路有两个人，击杀/助攻的收益是上路的两倍。而且下路离龙坑近，拿到人头后可以顺势控龙。Fizz 下路游走的标准路线是中路推线后走河道，如果有视野被发现了，可以绕下路三角草丛或者 E 翻墙绕后。

### 上路游走

上路的收益比下路小，但如果上路对手推线很深或者你的上单被压得很惨，一次成功的 gank 可以稳住上路局势。注意上路距离更远，来回的时间成本更高，要确保值得。

### 反野入侵

如果你知道对方打野在另一边（比如他刚出现在上路），你可以考虑入侵他的野区：偷他的蓝 buff/红 buff，或者在他野区插眼获取信息。这种操作风险较高但收益也大，适合你优势很大的局。

## 游走后的回线

成功的游走不只是"去拿人头"，还包括**高效回线**。拿到击杀后的理想流程是：拿人头 → 推掉对方下路塔皮/控龙 → 回城补给 → 回到中路。不要拿了人头就在下路晃悠不回去，你中路的兵和经验在流失。

::: warning 常见错误
游走失败后不要恋战。如果你到了下路发现没有机会，立刻走，不要硬蹲。蹲的每一秒你都在丢中路的兵线经验。
:::
