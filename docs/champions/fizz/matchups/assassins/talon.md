# vs 男刀 (Talon)

::: info 对局概览
**难度**：⭐⭐ 简单 | **对线体验**：节奏对抗 | **关键词**：他的游走 vs 你的单杀
:::

## 核心思路

男刀的对线能力不如你，但他的游走速度极快（E 翻墙）。这个对局的核心矛盾不是"谁打赢谁"，而是**他跑了你怎么办**。你能在对线压制他，但如果他放弃对线去游走拿到人头，优势就转移了。
