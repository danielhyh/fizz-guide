# vs 辛德拉 (Syndra)

::: info 对局概览
**难度**：⭐⭐⭐ 中等 | **对线体验**：前期挨打，6级反杀 | **关键词**：E 躲她的 E、6级时机

## 核心思路

辛德拉前期的消耗能力远强于你（Q 戳你、W 砸你、E 推球晕你），但她有一个致命弱点——**没有位移**。一旦她的 E（击退/眩晕）交了，她在接下来的十几秒内面对你的 all-in 几乎没有自保能力。6级后你的 R 对她来说是噩梦，因为她只能靠闪现来躲。
