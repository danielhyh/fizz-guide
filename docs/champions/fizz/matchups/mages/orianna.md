# vs 发条 (Orianna)

::: info 对局概览
**难度**：⭐⭐⭐ 中等 | **对线体验**：稳健对拼 | **关键词**：躲球、找进场角度
:::

## 核心思路

发条是一个非常稳健的法师，她的球可以同时提供消耗和区域控制。对线她的关键是**始终注意球的位置**，不要站在球的路径上白吃伤害。6级后你有击杀她的能力，但要小心她的 R 可以在你 Q 进去的瞬间把你拉回来。
