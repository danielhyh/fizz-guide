# vs 拉克丝 (Lux)

::: info 对局概览
**难度**：⭐⭐ 简单 | **对线体验**：优势对局 | **关键词**：躲 Q 光束、全力击杀
:::

## 核心思路

拉克丝是 Fizz 的典型优势对局。她的一切自保能力都依赖于 Q（光之束缚）命中你，而你的 E 可以完美躲掉她的 Q。只要她 Q 没命中，她在你面前就是一个没有位移、没有自保的脆皮。3级开始就可以找机会换血，6级基本就是击杀线。
