# vs 加里奥 (Galio)

::: info 对局概览
**难度**：⭐⭐⭐⭐⭐ 极难 | **对线体验**：系统克制 | **关键词**：放弃单杀、转游走
:::

## 核心思路

加里奥是 Fizz 最难打的对局之一，可以说是系统级克制。他天生高魔抗、W 的被动护盾吃你的法术伤害、主动 W 的嘲讽可以打断你的连招节奏。你基本不可能在对线中单杀一个合格的加里奥。

**这个对局的正确思路不是"怎么打赢他"，而是"怎么让这条线的劣势不影响全局"。** 具体策略是尽量接好兵、不送人头、找机会游走其他路。你游走的时候他也会用 R 支援，但你的游走频率可以比他高。
