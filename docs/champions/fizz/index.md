# 🐟 小鱼人 (Fizz)

::: tip 我的本命英雄
Top1主力英雄，一路打上来的老伙计。下面是我对这个英雄的全部理解。
:::

## 英雄定位

Fizz 是一个 **AP 刺客型中单**，核心价值是在前中期利用极高的单体爆发秒杀对方 C 位，滚雪球，同时凭借 E 技能的无敌帧和位移能力实现进场后的生存。他不是一个靠持续输出打团的英雄，而是一个靠**精准时机选择**和**一套技能决定胜负**的刺客。

## 我对这个英雄的核心理解

玩 Fizz 最容易掉进的坑是"觉得自己是刺客就要一直打"。实际上 Fizz 的节奏感比操作本身更重要——什么时候该忍、什么时候该打、什么时候该跑去带节奏，这些判断的质量决定了你的段位。

**前期不是用来赢线的，是用来不亏的。** 1-2 级老老实实接兵，3级开始有选择性地短换血，6级才是你真正的发力点。过了6级之后，Fizz 的游戏节奏就应该变成"推线 → 游走 → 回线"的循环，而不是一直跟对手在中路对戳。

## 知识体系导航

**基础体系** — 连招逻辑、强势期节奏、符文出装，这些是所有对局的基石。如果你刚开始学 Fizz，先把这部分吃透。

**对线攻略** — 按英雄分类的对线细节。每个对局都有不同的打法重点，核心是理解"这个对手的弱点在哪、我的窗口期在哪"。

## 英雄池备注

Fizz 并不是万金油，遇到加里奥这种系统克制的时候非常难受。未来可能会扩展的备选英雄方向：

- **阿卡丽 (Akali)** — 同为 AP 刺客，但对线能力更强，可以作为 Fizz 被 ban 时的替代
- **加里奥 (Galio)** — 反向思路，当对面先选了刺客的时候可以用来反制
- **卡牌大师 (Twisted Fate)** — 完全不同的风格，强调全图支援而非单杀，拓宽英雄池思路
