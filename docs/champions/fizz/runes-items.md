# 符文出装

## 符文选择思路

符文不是一套固定模板走天下，而是根据对局情况做调整。但 Fizz 的符文选择空间相对集中，核心就是**电刑 vs 征服者**的选择。

## 主系：主宰系（绝大多数对局）

### 电刑（Electrocute）

**适用场景**：几乎所有对局的默认选择

电刑和 Fizz 的 Q + W 短换血模式完美契合——Q 进去 + W 平A + 再补一下平A 就能触发电刑，配合海蜇毒素的持续伤害，一套换血量非常可观。

副系选择：
- **血之滋味**：换血后回血，提升对线续航
- **眼球收集器**：滚雪球提供额外法强
- **贪欲猎手 / 灵巧猎手**：全能吸血 vs 物品 CD，看个人偏好

### 备选：征服者（Conqueror）

**适用场景**：对面多坦克、需要打持久战的局

征服者的 Fizz 打法完全不同——你不再是一套秒人的刺客，而是靠持续输出慢慢磨。这套符文在对面有 2-3 个坦克/半肉的时候有奇效，但需要你改变打法习惯。

## 出装体系

### 核心出装路线

**卢登的激荡 → 法术穿透鞋 → 中娅沙漏**

这是最稳定的出装路线。卢登提供法穿和波次伤害，法穿鞋进一步降低对手魔抗，中娅给你 all-in 之后的生存空间。

### 出装选择逻辑

出装的决策不是"我该出什么"，而是"当前局势需要我解决什么问题"：

- **对手血量高/出了魔抗** → 虚空之杖（百分比法穿）
- **需要更多爆发来秒人** → 灭世者的死帽
- **对面有致命 CC** → 中娅沙漏提前出（甚至第一件）
- **对面有强力治疗** → 莫雷洛秘典

### 鞋子选择

法穿鞋是 90% 对局的选择。唯一考虑换的情况：
- 对面全AD + 多控 → 护甲鞋
- 对面有极强 AP 威胁 + 多控 → 水银鞋

::: tip 出装心法
不要迷信"版本最强出装"，要看你这把游戏的具体局势。同样是对面有魔抗，对手只出了一个小魔抗斗篷和出了满级水银饰带，你的应对完全不同。
:::

---

> 💡 出装体系会随版本更新调整，留意每次大版本的装备改动。
